\subsubsection*{Description}

This library provides the function of D\+AC (Digital to Analog Converter) by using the P\+W\+M0 of Timer Counter 0, 1 and 2. When {\ttfamily R\+T\+\_\+\+D\+A\+C\+\_\+analog\+Write} is used, the frequency of the corresponding timer is changed. If you want to custom the timer frequency, please change it later.
